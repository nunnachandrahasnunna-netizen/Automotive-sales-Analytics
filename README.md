# Automotive-sales-Analytics
Automotive Sales Dashboard Using Excel
| Column Name    | Description                        |
| -------------- | ---------------------------------- |
| Transaction ID | Unique ID for each sale            |
| Date           | Date of sale                       |
| Car Model      | Name of vehicle sold               |
| Brand          | Manufacturer (e.g., Tata, Hyundai) |
| City           | Location of sale                   |
| State          | State name                         |
| Salesperson    | Person responsible for sale        |
| Units Sold     | Number of cars sold                |
| Revenue        | Total sales value                  |
| Cost           | Cost incurred                      |
| Profit         | Revenue - Cost                     |
