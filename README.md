# Automotive-sales-Analytics
Automotive Sales Dashboard Using Excel
🚗 Automotive Sales Analytics Project (India)

A complete data analytics project focused on analyzing four-wheeler automotive sales in India, providing insights into revenue, profit, sales performance, and regional trends using Excel dashboards.

📌 1. Project Overview

This project analyzes a structured dataset of automobile sales across India to uncover meaningful business insights.

🎯 Objective:

The main goal is to:

Understand sales performance
Track revenue & profit trends
Identify top-performing regions & salespersons
Support data-driven decision-making
🧠 What this project solves:
Which region generates the most revenue?
Which salesperson performs best?
Which car models are most profitable?
How cost impacts profit margins?
📊 2. Data Summary

This dataset contains:

Total Records: 100 rows (transactions)
Industry: Automotive (Four-Wheeler Sales)
Geography: India (Multiple cities/states)
Data Type: Structured (Excel)
📈 Data Nature:
Transaction-level data
Business metrics (Revenue, Cost, Profit)
Categorical + Numerical mix
📋 3. Dataset Columns (Detailed Explanation)
Column Name	Description
Transaction ID	Unique ID for each sale
Date	Date of sale
Car Model	Name of vehicle sold
Brand	Manufacturer (e.g., Tata, Hyundai)
City	Location of sale
State	State name
Salesperson	Person responsible for sale
Units Sold	Number of cars sold
Revenue	Total sales value
Cost	Cost incurred
Profit	Revenue - Cost
🔍 Column Logic:
Profit = Revenue – Cost
Helps measure business efficiency
Useful for identifying loss-making vs profitable sales
🔍 4. Key Insights (Detailed)

After analyzing the dataset:

📈 Sales Trends:
Certain cities consistently generate higher revenue
Metro cities often dominate in sales volume
💰 Profitability:
High revenue ≠ high profit (due to cost variations)
Some models have low margins but high sales
🧑‍💼 Salesperson Performance:
Top performers contribute significantly to revenue
Helps in performance evaluation
🚗 Product Insights:
Some brands dominate market share
Specific car models drive most of the profit
🌍 Regional Insights:
States like Maharashtra, Karnataka, Tamil Nadu often perform better
Regional demand varies based on affordability & preference
